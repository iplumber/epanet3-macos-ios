# E3 UI 稳定性回归清单 v1

## 1. 使用说明
- 本清单用于 E3 第二版人工回归（macOS UI 交互稳定性）。
- 执行顺序建议：`net1.inp` -> `any1.inp` -> `400.inp`。
- 每项结果记录：`通过 / 失败 / 备注`。

## 2. 回归环境
- 平台：macOS（Xcode Run）
- 工程：`EPANET3App/EPANET3Xcode/EPANET3Xcode.xcodeproj`
- 算例：
  - `epanet  resource/example/Epanet 管网/net1.inp`
  - `epanet  resource/example/Epanet 管网/any1.inp`
  - `epanet  resource/example/可计算 算例管网 inp/400.inp`

## 3. 基础交互稳定性
- [ ] 打开 `.inp` 后画布正常显示，无崩溃
- [ ] 滚轮缩放连续 20 次，无异常跳变
- [ ] 中键/拖拽平移连续 20 次，无卡死
- [ ] 双击重置视图可用
- [ ] Esc 取消选中可用
- [ ] 鼠标坐标显示稳定更新

## 4. 编辑流程稳定性（D1/D2）
- [ ] 选中节点后属性面板打开，字段可读
- [ ] 节点属性保存后，画布与属性值同步更新
- [ ] 管段属性保存后，画布与属性值同步更新
- [ ] 新增节点成功后自动聚焦并高亮
- [ ] 新增管段成功后自动聚焦并高亮
- [ ] 删除当前选中对象可用
- [ ] 按 ID 删除节点可用
- [ ] 按 ID 删除管段可用

## 5. 参数与单位稳定性（D3）
- [ ] 参数区“刷新/保存”可用（Trials/Accuracy/DemandMult/Duration/HydStep/ReportStep）
- [ ] Flow Units `GPM -> LPS` 切换后重载成功
- [ ] Flow Units `LPS -> GPM` 切换后重载成功
- [ ] 切换单位后可继续编辑/新增/删除

## 6. 计算与结果稳定性（D4）
- [ ] 点击运行后成功完成，无崩溃
- [ ] 结果上图模式“无/压力/流量”切换稳定
- [ ] 压力上图颜色与图例范围显示一致
- [ ] 流量上图颜色与图例范围显示一致
- [ ] 选中高亮与结果着色可同时工作

## 7. 错误提示可用性（E2 复核）
- [ ] 非法输入时出现可读错误提示
- [ ] 错误提示含错误码与接口/对象上下文
- [ ] 错误对象可自动定位并打开属性面板

## 8. 压力回归建议
- [ ] 连续执行 10 次“编辑 -> 保存 -> 运行计算”
- [ ] 连续执行 10 次“切换结果上图模式”
- [ ] 连续执行 10 次“Flow Units 双向切换（GPM/LPS）”
