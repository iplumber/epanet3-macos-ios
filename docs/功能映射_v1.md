# 功能映射 v1（原型到实现任务）

## 1. 说明
- 目标：把 `UI` 原型功能项映射为可开发任务。
- 状态定义：
  - 已实现：当前代码可稳定使用
  - 部分实现：有基础能力，但未形成完整闭环
  - 待开发：当前无可用实现
  - 非范围：v1 不做
