# 验收标准 v1

## 1. 范围约束
- 仅支持 `.inp`
- 不支持 `.net`

## 2. 功能验收项

### 2.1 文件管理
- 可打开 `.inp`
- 可保存与另存 `.inp`
- 打开失败有明确错误提示

通过标准：
- `net1.inp`、`any1.inp`、`400.inp` 均可打开

### 2.2 对象操作（v1 最小集）
- 可选中对象并查看属性
- 可编辑节点和管段核心属性
- 可删除单个对象并保持模型有效

通过标准：
- 编辑后保存再打开，数据一致

### 2.3 参数设置
- 可设置单位制（英制/公制）
- 可设置计算时长、步长、水损公式

通过标准：
- 参数变化后计算结果发生预期变化

### 2.4 计算与反馈
- 可运行计算
- 成功时有明确反馈（状态栏或弹层）
- 失败时有错误码和可读提示

通过标准：
- 三算例运行流程稳定，无崩溃

### 2.5 结果显示
- 至少支持压力、流量两类结果上图
- 选中对象可查看对应结果值

通过标准：
- 结果图层切换可用，图例显示正确

## 3. 回归算例
- `epanet  resource/example/Epanet 管网/net1.inp`
- `epanet  resource/example/Epanet 管网/any1.inp`
- `epanet  resource/example/可计算 算例管网 inp/400.inp`

## 4. 非功能验收
- 主流程不崩溃
- 基础交互无明显卡顿
- 错误提示不出现“无上下文”信息
