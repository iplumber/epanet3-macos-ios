# EPANET macOS/iOS 发布说明 v1

## 1. 版本信息
- 版本名称：v1
- 发布阶段：主线阶段 A/B/C/D/E 已完成
- 适用平台：macOS、iOS
- 文件格式支持：仅 `.inp`（不支持 `.net`）

## 2. 本版目标
- 建立从加载算例到结果查看的最小可用闭环。
- 完成引擎、桥接、界面、回归四层协同。
- 提供可读可定位的错误信息，支撑后续界面强化与版本迭代。

## 3. 已交付能力
- 文件流程：打开、编辑、保存/另存 `.inp`（最小集）。
- 对象编辑：节点/管段核心属性编辑，新增/删除（含按 ID 删除）。
- 计算流程：运行计算、成功/失败反馈、状态同步。
- 结果可视化：压力/流量结果上图与图例展示。
- 参数设置：常用计算参数可读写，Flow Units 支持 `GPM/LPS` 重载切换。
- 错误体验：错误码 + 可读文本 + 接口/对象/参数上下文，可用于 UI 定位。

## 4. 回归与稳定性
- 三算例回归通过：`net1.inp`、`any1.inp`、`400.inp`
- 大算例稳定性检查：`10000.inp`、`90000.inp`
- CLI 错误可用性自检通过：`--self-check-errors`
- UI 稳定性回归清单：`docs/E3_UI稳定性回归清单_v1.md`

## 5. 已知限制
- Flow Units 切换当前采用“临时副本重载”策略，不是原文件原位改写。
- 结果上图当前为最小集（压力/流量），更多指标待后续扩展。
- iOS 小屏交互仍有优化空间（布局密度与触控效率）。

## 6. 升级与兼容说明
- 旧工程迁移后，建议优先确认 `.inp` 主流程（打开 -> 运行 -> 结果）可用。
- 本版本不包含 `.net` 导入、导出与兼容转换。
