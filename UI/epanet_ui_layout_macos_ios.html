macOS — 窗口布局
分区：菜单栏 · 工具栏 · 画布 · 右侧 Inspector · 底部状态栏
Inspector 常驻，选中对象实时刷新，不打断操作流 （参考 Pixelmator Pro 风格）
计算面板：独立浮动窗口或 Panel，不占主窗口空间
图表：可独立窗口平铺，支持 macOS Stage Manager
iOS — 页面布局
分区：导航栏 · 全屏画布 · 底部工具条（Safe Area 上方）
属性编辑：点击对象弹出底部 Sheet，半屏展示常用属性，上滑全屏 （iOS 16 UISheetPresentationController）
参数设置：进入独立设置页面，原生 Form 样式
横屏：底部工具条改为左侧竖排，画布占满宽度
