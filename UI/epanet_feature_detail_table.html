功能项 功能描述 macOS 实现 iOS 实现
— 文件管理
新建 / 打开 新建空白管网；打开 .inp 文件；支持最近文件列表 macOSFile 菜单 + ⌘N / ⌘O；支持拖拽到 Dock iOS启动页选择；Files App 集成
保存 / 另存为 保存为 .inp 原生格式；另存为时可选格式 macOS⌘S / ⌘⇧S；iCloud Drive 自动同步 iOS导航栏按钮；Sheet 选择存储位置
导入 .inp 导入标准 .inp 文本格式并校验结构完整性 共用仅支持 .inp，统一解析器，两端相同体验
导出报告 导出 PDF 计算报告；含管网概览图 + 计算结果汇总表 macOS打印对话框 + 存为 PDF iOSShare Sheet 分享 / 存档
— 管网对象（增删改查）
添加对象 节点 / 水池 / 水库 / 管段 / 阀门 / 水泵六类对象 macOS工具栏选择对象类型，点击画布放置 iOS底部工具条选类型，点击/拖拽放置
编辑属性 选中对象后编辑全部水力属性；支持多选批量修改部分属性 macOS右侧 Inspector 侧边栏（常驻） iOS点击对象弹出底部 Sheet（半屏/全屏）
移动 / 删除 拖动节点重新布局；Delete 键或菜单删除；关联管段自动处理 macOS拖拽 + Delete 键；右键菜单 iOS拖拽移动；长按 Context Menu 删除
查询 / 搜索 按 ID 或名称快速定位对象；高亮并居中显示 macOS⌘F 触发搜索框，结果列表点击跳转 iOS导航栏搜索按钮
管段绘制 点击起点节点，点击终点节点完成管段；支持折点 macOS点击起点→移动→点击终点；ESC 取消 iOS选管段工具→点击→点击；单指操作
— 视图控制
缩放 连续缩放；快速适配全图；100% 原始比例 macOS滚轮缩放；⌘+ / ⌘-；⌘0 适配全图 iOS双指捏合；双击适配全图
平移 拖动画布平移视图 macOS空格+拖拽；或双指触控板滑动 iOS单指拖拽（非选中状态）
全屏 最大化画布区域，隐藏辅助面板 macOS系统全屏按钮；⌃⌘F iOS横屏时自动扩展；隐藏底部栏按钮
— 显示样式
图例设置 定义压力 / 流量等结果的颜色分级区间；可自定义分段数和色带 共用图例面板；实时预览色带变化
对象样式 节点半径 / 管段线宽 / 颜色可单独或按类型统一设置 macOSInspector 内样式区域 iOS属性 Sheet 内样式折叠区
文字标注 控制显示 ID / 名称 / 高程 / 计算结果值等叠加标注；按缩放级别自动显隐 共用视图菜单 / 工具栏 Toggle 按钮
背景图层 （可选）加载本地图片或在线底图作为参考背景 共用图层面板控制透明度与显隐；v1.0 可推迟
— 参数设定
单位制 SI（L/s·m·kPa）或 US 惯用单位（GPM·ft·psi）全局切换；所有输入输出实时换算 共用设置窗口顶部，切换时提示重新验证数值
水力参数 最大迭代次数；收敛精度（水头/流量）；压降单位；粘度温度修正 macOS偏好设置窗口 → 水力标签页 iOS设置 → 水力参数 Sheet
水损公式 Hazen-Williams / Darcy-Weisbach / Chezy-Manning 三选一 共用计算参数面板内 Segmented Control
模拟时长 稳态（Steady State）或时变模拟（EPS）；设定总时长、水力步长、报告步长 共用计算面板；稳态下时间项自动折叠
— 运行计算
执行计算 一键触发；后台线程运行，不阻塞 UI；进度指示 macOS工具栏「运行」按钮 + ⌘R；状态栏进度 iOS底部主按钮；运行中显示 Activity Indicator
成功提示 计算完成：显示用时、迭代次数，结果自动叠加到画布 macOS状态栏绿色标记 + Toast iOS顶部短暂 Banner
错误报告 网络不连通 / 负压 / 不收敛等错误；精确定位到问题对象；区分警告与错误 macOS错误面板列表；点击条目高亮定位对象 iOSSheet 弹出错误列表；同样支持点击定位
— 结果查看
图形渲染结果 压力 / 流量 / 流速 / 水头等结果以色带热力图叠加在管网图上 共用图例面板切换结果变量；实时重渲染
对象结果查看 点击单个节点 / 管段查看所有计算结果数值 macOSInspector 下方结果区域 iOS属性 Sheet 结果标签页
时序图表（EPS） 时变模拟结果：对象某属性随时间变化的折线图；支持多对象叠加对比 macOS独立图表窗口（可并排） iOS全屏 Sheet 内图表；可手势滑动时间轴
结果汇总表 节点/管段全量结果的可排序表格；支持复制到剪贴板 macOS结果面板标签页，可调整列宽 iOSSheet 内横屏优化表格
